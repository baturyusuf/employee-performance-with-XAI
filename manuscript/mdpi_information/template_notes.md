# Template Notes

## Template Status

Official MDPI template files were not committed into this public repository.

Verified official sources on 2026-06-25:

- MDPI LaTeX author page: https://www.mdpi.com/authors/latex
- *Information* instructions for authors: https://www.mdpi.com/journal/information/instructions
- Official MDPI Overleaf template listing: https://www.overleaf.com/latex/templates/mdpi-article-template/fcpwsspfzsph

The *Information* instructions state that authors are encouraged to use the Microsoft Word or LaTeX template and that LaTeX submissions should be collated into a ZIP folder including source files and images. The instructions also include template-use limitations. Because this repository is public, the official MDPI template class and support files were not downloaded or redistributed here.

## Current Manuscript Source

`main.tex` is written in MDPI-style structure and uses the expected MDPI metadata commands where practical:

- `\Title`
- `\Author`
- `\address`
- `\corres`
- `\abstract`
- `\keyword`
- back-matter declarations
- `\bibliography{references}`

It expects the official MDPI `Definitions/` folder and class file to be supplied before PDF compilation. Without those files, local LaTeX compilation will not succeed.

## Manual Steps Before Submission

1. Download the current official MDPI LaTeX template from MDPI or Overleaf.
2. Confirm that the journal option for *Information* is current.
3. Copy or merge the content of `main.tex` into the official template.
4. Add the official `Definitions/` folder locally for compilation.
5. Complete all `AUTHOR_TO_COMPLETE` fields.
6. Verify figure sizing, table layout, and reference formatting under the current MDPI template.
7. Prepare the MDPI submission ZIP with `main.tex`, `references.bib`, figures, tables or supplementary files, and any required template support files.

## PDF/DOCX Status

No compiled PDF or DOCX is generated by this package unless a local TeX or conversion tool is available. Do not create a placeholder PDF.
